# Form-Builder-Go

Go语言版本的表单构建器，移植自PHP版本的FormBuilder。基于 [@form-create/element-ui](https://github.com/xaboy/form-create) 实现快速生成现代化的Form表单。

## 特点

- 使用JSON数据生成表单
- 支持扩展，生成任何Vue组件和HTML标签
- 支持组件之间联动
- 提供丰富的表单操作API
- 支持子表单和分组
- 高性能
- 类型安全的Go实现
- 支持泛型和现代Go最佳实践

## 环境需求

- Go >= 1.24

## 支持 UI

- ElementUI
- IView

## 安装

```bash
go get github.com/formbuilder-go
```
